Sublime Text 3 与 ST2 安装不同，不能使用命令行安装 Package Control，这里提供一下安装方法。

下载附件，将附件中的Package Control目录拷贝到插件目录（就是 `菜单->首选项->浏览程序包` 打开的目录）。

如果能看到 `菜单->首选项->Package Control` 菜单则说明安装成功。
